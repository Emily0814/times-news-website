### User Story
- 사용자는 랜딩 페이지에서 뉴스를 볼 수 있다.
- 각 뉴스는 이미지, 제목, 내용, 출처, 날짜를 포함한다.
- 사용자는 카테고리별로 뉴스를 검색할 수 있다. (예: 스포츠, 기술, 비즈니스 등)
- 사용자는 키워드로 뉴스를 검색할 수 있다.
- 웹사이트는 반응형 웹페이지로, 다양한 기기에서 사용할 수 있다.
- 사용자는 페이지네이션을 통해 더 많은 뉴스를 볼 수 있다.
